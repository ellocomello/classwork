CSS selectors

tag selector Use the tag name to select the word.
class You use a period to select a class.
id You use a # to select a specific id.
child To select a child you use the > symbol after an element.
Descendant To select a descendant you put a space after selecting your element.
Link You use element when selecting a link.
Visited You use a tag with a colon and put visited in it to style a visited link.
Hover You use the tag with a colon and put hover in it to style a Hover link.
Active You use the tag with a colon and put hover in it to style a active link.
Sibling You use a + to select a sibling.

CSS attributes

term definition
Absolute units Absolute units do not change with the screen and are not recommended for websites
CSS size font-relative units em and rem both refer to a font size, though they are often used to define other sizes in CSS. You’ll see both of them often so we’re going to explain both, but as a rule-of-thumb, prefer rem.
CSS attribute: font-size he font-size CSS property sets the size of the font. Changing the font size also updates the sizes of the font size-relative length units, such as em, ex, and so forth.
CSS size viewport relative units: a type of relative length unit that define a size based on the dimensions of the browser's viewport
CSS visibility The CSS visibility property defines whether an element is shown or hidden. The key characteristic is that hidden elements still affect the layout of the document by occupying their original space.
CSS attribute: display values: The CSS display property is a fundamental attribute that controls how an HTML element is rendered and how it interacts with other elements in the document's flow. It determines the type of box an element generates, which is crucial for page layout.
CSS attribute: font-family The CSS font-family property specifies the prioritized list of typefaces (fonts) to be used for the selected element's text.
CSS padding CSS padding is the space between an element's content (text, images, etc.) and its border.
CSS margin The CSS margin property defines the transparent space around an element, outside of any defined borders.
CSS border The CSS border properties are used to define the style, width, and color of an element's border, which is a line that wraps around the element's padding and content.
